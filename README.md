1 : IoC , 控制反转,  类本身不创建其依赖对象，而是由其他外键实体将其传入；
 IoC 控制反转也是 Java 7大设计原则之一 依赖倒置,单一职责的一种体现；
 
2: Ioc 中的Dependency Injection 依赖注入,注入方式有 : 
    2.1 构造注入
    2.2 set注入
    2.3 接口注入
    目的都是为了给目标类提供相应的实例对象;
